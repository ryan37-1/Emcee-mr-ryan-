## Features
- Elegant responsive design
- AI script generator and theme suggestions (via Gemini API)

## Setup Instructions
1. Clone this repository or upload your HTML files.
2. If you use Gemini AI features, add your API key in the index.html as instructed.
3. To make your site live, enable GitHub Pages: 
    - Go to settings → Pages
    - Select branch: `main`, folder: `/root`
    - Click "Save"
